# INVEN Portal MVP

Version 1 is a lightweight Firebase-backed client/admin portal for a small web development company.

## Client Portal

- Dashboard: project overview, progress, recent activity, and next payment due.
- Files: upload and download project files.
- Invoices: view invoice status and download PDFs.
- Revision Requests: submit title, description, priority, and optional file/image.
- Website Content: edit approved website fields only.
- Profile: basic account info and password change.

## Admin Portal

- Dashboard
- Files
- Invoices
- Revision Requests

Messages, live chat, notifications, analytics, settings, and unfinished placeholder pages have been removed from the V1 surface.

## Firebase Collections

- `users`
- `projects`
- `files`
- `invoices`
- `revisionRequests`
- `websiteContent`
